https://deybimorales.gitlab.io

Suceden cambios y este es el segundo desde que tengo presencia en internet, anteriormente estábamos en blogger, después no pasamos a Nikola con Github. Ahora vamos utilizar Pelican almacenado en Gitlab Pages.

Razones:

  • A Github se le reconoce por trabajar eficientemente con Jekyll. Gitlab expresa tener soporte a una extensa lista de Generadores de Páginas Web Estáticas cada uno con su versión para funcionar con Gitlab CI.

  • Gitlab genera el blog en la nube. No necesito compilar el blog en mi laptop y luego subirlo.

  • Gitlab tiene todo un editor de archivos. Por lo tanto puedo modificar el blog desde cualquier navegador. Me recuerda a blogger y wordpress.com.
